# UI Agent 任务：界面审查和优化

## 📋 任务说明

请阅读项目代码和设计，了解当前界面实现，提出优化建议。

## 📚 需要阅读的材料

1. **项目简报**: `AGENT_TEAM_BRIEF.md`
2. **前端代码**: `frontend/src/` 目录
   - `pages/` - 页面组件
   - `components/` - 通用组件
3. **样式文件**: `frontend/src/index.css`, Tailwind 配置
4. **当前问题**: 已知的 UI/UX 问题清单

## 🎯 审查重点

1. **视觉设计**
   - 色彩搭配
   - 字体大小和层级
   - 间距和布局

2. **用户体验**
   - 交互流程是否流畅？
   - 反馈是否及时？
   - 错误提示是否友好？

3. **响应式设计**
   - 移动端适配
   - 不同屏幕尺寸适配
   - 触摸操作优化

4. **可访问性**
   - 色彩对比度
   - 键盘导航
   - 屏幕阅读器支持

## 📝 输出要求

请提供一份 UI 审查报告，包含：
- 当前界面评估
- 发现的问题
- 优化建议（附设计稿或代码示例）
- 优先级排序

## ⏰ 完成时间

请在阅读完成后输出报告。
