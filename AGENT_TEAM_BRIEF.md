# Course AI Helper - Agent Team 项目简报

## 🎯 项目概述

**项目名称**: Course AI Helper (AI 课程学习助手)

**项目目标**: 为教育机构提供视频课程学习辅助系统，支持视频管理、字幕生成、内容总结、知识点搜索等功能。

**技术栈**:
- **前端**: React + TypeScript + TailwindCSS + Vite
- **后端**: Python + FastAPI + SQLAlchemy (Async)
- **数据库**: SQLite (主数据) + ChromaDB (向量检索)
- **AI 服务**: 
  - ASR: Faster-Whisper (语音转文字)
  - LLM: Ollama (qwen2.5:7b) / 阿里云百炼
  - Embedding: Jina AI
